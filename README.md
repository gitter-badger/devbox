# DEVBOX

A Fullstack Js ready Docker image, built on top of Debian Jessie.

### Usage example

    sudo docker run -it --name devbox -p3000:3000 mikamboo/devbox

### Installed apps

* NODE : v0.10.38
* UTIL : git, build-essential, python
* NPM  : bower, strongloop, json-server, yo, gulp, grunt
